box-shadow实现边框

发表于 2016-06-07

今天看了魔法哥的一(新)次(书)讲(推)座(销)视频,发现了一个使用box-shadow实现边框的很有意思的方法

Demo

div{
  width:300px;
  height:300px;
  margin: 50px auto;
  box-shadow:0 0 0 5px #888888,0 0 0 10px #FF8800,0 0 0 15px #FFBB33;
}
div:nth-child(1){
  background:#0099CC;
}
div:nth-child(2){
  background:#AA66CC;
  border-radius: 150px;
}
