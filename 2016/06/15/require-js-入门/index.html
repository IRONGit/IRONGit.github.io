require.js 入门

发表于 2016-06-15

加载require.js

<script src="js/require.js" data-main="js/main.js"></script>

这条语句中的data-main定义了程序的主模块,类似于oop语言中的main函数

对模块加载的自定义

为了方便,可以为引入的模块定义别名,比如

requirejs.config({
  paths:{
    jquery:'jquery-1.11.3'
  }
})

这段代码为js目录下的jquery-1.11.3.js文件定义的别名为jquery(不用写.js后缀)

当然,如果js文件在别的目录中,还可以改变根目录

requirejs.config({
    baseUrl:"js/baseJs",
    paths:{
      jquery:'jquery-1.11.3'
    }
});

自定义模块

//在myModule.js中
define(['jquery'],function($){
    function myModule(){

    };
    myModule.prototype.method=function(){

    }
    return{
      myModule:myModule
    }
});

可以通过define()自定义模块,第一个参数是一个数组,代表引入的模块,第二个参数是一个匿名的回调函数,

匿名函数的参数的顺序是引入模块的引用的顺序

使用模块

requirejs(['jquery','myModule'],function($,mm){
  new mm.myModule().method();
});
