RegExp类型

创建正则表达式的两种方式

1.使用字面量形式

var expression=/pattern/flags

例如:
var pattern1=/at/g
匹配字符串中所有”at”的实例

每个正则表达式都可以带有一个或多个flags
g:表示全局(global)模式
i:表示不区分大小写(case-insensitive)模式
m:表示多行模式(multiline)模式

2.使用RegExp构造函数

var pattern2=new RegExp("cat",g);
